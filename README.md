# Cranial Hemorrhage

A brutal, dark-themed website for the solo death metal project Cranial Hemorrhage, unleashing unrelenting fury from the abyss.

## Features

- **Dark Metal Design**: Black backgrounds with red/gold accents, brutal typography
- **Responsive**: Fully responsive across all devices
- **Aggressive Animations**: Intense CSS animations and transitions
- **Embedded Music Players**: YouTube iframes for music videos and audio tracks
- **Photo Gallery**: Metal-themed imagery grid
- **Contact Form**: Functional contact form for inquiries
- **Raw Typography**: Metal Mania font for that brutal look

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Metal Mania)

## Sections

1. **Hero**: Aggressive header with project name and call-to-action
2. **About**: Project story and creative process
3. **Music**: Embedded video players, audio tracks, and discography
4. **Gallery**: Photo gallery with metal imagery
5. **Contact**: Contact form and project information
6. **Footer**: Links and social media

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. For local development, run a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Visit `http://localhost:8000` in your browser

## Customization

- **Colors**: Red (#ff0000) and gold (#ffd700) accents on black background
- **Fonts**: Metal Mania for brutal typography
- **Content**: Update project info and discography in HTML
- **Images**: Replace Unsplash URLs with actual project photos
- **Music**: Update YouTube embed URLs and audio sources with real content
