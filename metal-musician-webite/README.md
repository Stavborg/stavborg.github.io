# Metal Musician Website

This project is a personal website for a metal musician's solo project. It serves as a platform to showcase the musician's work, including music, upcoming shows, and a way for fans and industry professionals to get in touch.

## Project Structure

The project is organized as follows:

```
metal-musician-website
├── index.html          # Main entry point of the website
├── about.html          # Biography and background of the musician
├── music.html          # Music showcase with audio samples and discography
├── shows.html          # Upcoming shows and events
├── contact.html        # Contact form and social media links
├── css
│   ├── styles.css      # Main stylesheet for layout and design
│   └── responsive.css   # Styles for responsive design
├── js
│   ├── main.js         # Main JavaScript functionality
│   └── audio-player.js  # Audio player functionality
├── assets
│   ├── images
│   │   ├── band-photos  # Photos of the band
│   │   ├── album-covers  # Album cover images
│   │   └── logos        # Logos for branding
│   ├── audio
│   │   ├── samples      # Audio samples
│   │   └── full-tracks  # Full music tracks
│   └── videos           # Music videos and performance clips
├── media
│   └── press-kit        # Press kit materials for media use
└── README.md            # Project documentation
```

## Features

- **Homepage**: A welcoming introduction with featured music.
- **About Page**: Detailed biography and influences of the musician.
- **Music Page**: Embedded audio samples and links to full tracks.
- **Shows Page**: Information on upcoming shows and events.
- **Contact Page**: A form for fans and industry professionals to reach out.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open `index.html` in a web browser to view the website.
3. Customize the content in the HTML files as needed.
